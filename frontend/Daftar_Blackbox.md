# Daftar Blackbox Testing

## Kategori Pengujian

Dokumen ini mengorganisir checklist pengujian blackbox ke dalam kategori-kategori berikut:

1. **Halaman dan Komponen Utama**: Fitur inti aplikasi termasuk halaman pemasaran, pusat bantuan, dan dasbor
2. **Tampilan Data**: Fitur terkait tampilan data dan variabel, termasuk menu kontekstual
3. **Operasi Berkas**: Modal impor, ekspor, buka berkas, dan cetak
4. **Operasi Penyuntingan**: Fitur penyuntingan data termasuk find/replace, go to, dan fungsi clipboard
5. **Manajemen Data**: Modal untuk mengelola dan memanipulasi data
6. **Analisis Statistik**: Modal untuk analisis statistik deskriptif
