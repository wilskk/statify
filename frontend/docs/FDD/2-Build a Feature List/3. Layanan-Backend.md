# Area Fitur Utama: Layanan Backend

Area fitur ini mencakup layanan di sisi server dan interaksi API. Serupa dengan infrastruktur klien, fitur-fitur ini bersifat teknis dan tidak terlihat langsung oleh pengguna. Mereka berfungsi sebagai 'tulang punggung' yang memungkinkan fitur-fitur klien seperti mengunggah atau menyimpan file .sav. Ini adalah contoh lain dari daftar fitur 'System Interaction (SI)'.

| No. | Feature Set (Aktivitas) | Feature (Fitur) | Cross-reference |
| :--- | :--- | :--- | :--- |
| 1 | Pemrosesan File .sav (Server) | Memproses file .sav yang diunggah untuk mengekstrak data dan metadata | `@/backend/server/services/savService.ts` |
| 2 | | Membuat file .sav baru dari data dan variabel yang dikirim oleh klien | `@/backend/server/services/savService.ts` |
| 3 | API (Server) | Mendefinisikan endpoint API untuk operasi unggah dan pembuatan file .sav | `@/backend/server/routes/savRoutes.ts` |
| 4 | | Menangani permintaan HTTP untuk operasi file .sav | `@/backend/server/controllers/savController.ts` |
| 5 | Layanan API (Klien) | Menyediakan layanan klien untuk berinteraksi dengan API backend .sav | `@/services/api/savService.ts` |
| 6 | | Menyediakan konfigurasi untuk layanan API (URL, header, dll) | `@/services/api/config.ts` |
