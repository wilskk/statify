# Area Fungsional Utama: Analisis Statistik Deskriptif

Dokumen ini mendefinisikan cakupan fungsional untuk fitur-fitur analisis statistik deskriptif. Tujuan utama dari area fungsional ini adalah untuk menyediakan serangkaian perangkat analisis yang memungkinkan pengguna meringkas, memahami, dan mendapatkan wawasan awal dari data mereka.

Sesuai dengan metodologi *Feature-Driven Development* (FDD), seluruh fitur diorganisir ke dalam beberapa set fitur utama. Setiap set fitur direpresentasikan oleh dialog antarmuka pengguna yang spesifik, yaitu Frequencies, Descriptives, Explore, dan Crosstabs. Pendekatan ini memastikan bahwa setiap dialog memberikan nilai bisnis yang jelas dan terukur bagi pengguna.

> Catatan: Jalur berkas yang disebutkan dalam dokumen ini menggunakan alias `@/` yang merepresentasikan direktori root dari proyek.

---

## 1. Dialog Frequencies

| No  | Fitur                      | Deskripsi Fitur                                                                                                            |
| :-- | :------------------------- | :------------------------------------------------------------------------------------------------------------------------- |
| 1   | Analisis Frequencies       | Menyajikan analisis frekuensi dan distribusi untuk variabel kategorikal.                                                   |
| 2   | Pemilihan Variabel         | Memungkinkan pengguna memilih satu atau lebih variabel untuk dianalisis.                                                   |
| 3   | Tendensi Sentral           | Menyediakan opsi konfigurasi untuk statistik tendensi sentral, mencakup Mean, Median, Modus, dan Sum.                        |
| 4   | Dispersi                   | Menyediakan opsi konfigurasi untuk statistik dispersi, seperti deviasi standar, varians, rentang, nilai minimum, nilai maksimum, dan S.E. mean. |
| 5   | Distribusi                 | Menyediakan opsi konfigurasi untuk statistik distribusi, yaitu Skewness dan Kurtosis.                                      |
| 6   | Nilai Persentil            | Menyediakan opsi untuk menampilkan nilai persentil, termasuk kuartil, titik potong untuk 10 kelompok, dan persentil kustom.  |
| 7   | Urutan Tampilan            | Mengatur urutan tampilan output berdasarkan daftar variabel, urutan alfabetis, nilai menaik, atau nilai menurun.             |
| 8   | Tipe Diagram               | Memungkinkan pembuatan visualisasi data dalam bentuk diagram batang, diagram lingkaran, atau histogram.                      |
| 9   | Nilai pada Diagram         | Mengatur nilai yang ditampilkan pada diagram, baik dalam bentuk frekuensi maupun persentase.                               |
| 10  | Penyimpanan Otomatis       | Menyimpan pengaturan formulir secara otomatis untuk memudahkan penggunaan kembali pada sesi berikutnya.                      |
| 11  | Reset Pengaturan           | Menyediakan fungsi untuk mengembalikan semua pengaturan pada dialog ke kondisi awal.                                       |
| 12  | Tur Interaktif             | Menyediakan panduan interaktif yang menjelaskan langkah-langkah penggunaan fitur Frequencies.                                |
| 13  | Analisis Latar Belakang    | Menjalankan proses analisis pada *background thread* (web worker) untuk menjaga responsivitas antarmuka pengguna.            |
| 14  | Log Perintah SPSS          | Menghasilkan log perintah yang kompatibel dengan sintaks SPSS untuk tujuan dokumentasi dan reprodusibilitas.                 |
