## 2. Implementation

- **Lokasi Kode Sumber Utama:**
  - `frontend/components/Modals/File/Print/PrintOptions.tsx`: Komponen UI untuk opsi pencetakan.
  - `frontend/hooks/usePrintLogic.ts`: Hook yang menangani state dan logika UI.
  - `frontend/services/data/pdfPrintService.ts`: Service yang bertanggung jawab untuk pembuatan PDF.
- **Dependensi Kunci:**
  - `jsPDF`: Library untuk membuat dokumen PDF.
  - `jspdf-autotable`: Plugin `jsPDF` untuk membuat tabel.
  - `zustand`: untuk mengakses data, metadata, dan hasil analisis dari stores.
