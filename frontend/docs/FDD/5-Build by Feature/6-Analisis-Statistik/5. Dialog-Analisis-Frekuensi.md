## 1. Work Package

- **Fitur yang Dibangun:**
  - Mengimplementasikan dialog bertab untuk analisis Frekuensi.
  - Menyediakan tab untuk memilih statistik (persentil, tendensi sentral) dan grafik (bar, pie, histogram).
  - Membuat web worker (`frequencies.worker.js`) untuk menghitung tabel frekuensi dan statistik.
  - Mengintegrasikan `ChartBuilder` untuk membuat grafik dari hasil analisis.

## 2. Implementation

- **Lokasi Kode Sumber Utama:**
  - `frontend/components/Modals/Analyze/Descriptive/Frequencies/`: Direktori yang berisi komponen UI.
  - `frontend/hooks/useFrequencies.ts`: Hook yang mengelola state UI dan orkestrasi analisis.
  - `frontend/public/workers/Frequencies/frequencies.worker.js`: Worker untuk perhitungan tabel frekuensi.
  - `frontend/utils/chartBuilder/`: Utilitas untuk membangun komponen grafik dari data.
  - `frontend/stores/useOutputStore.ts`: Store untuk menampung hasil analisis.
