## 1. Work Package

- **Fitur yang Dibangun:**
  - Mengimplementasikan dialog bertab untuk analisis Tabulasi Silang (Crosstabs).
  - Menyediakan tab untuk pemilihan variabel (Baris, Kolom) dan opsi sel (Counts, Percentages).
  - Membuat web worker (`crosstabs.worker.js`) untuk menghitung tabel kontingensi dan statistik terkait.
  - Mengirimkan hasil output dalam format tabel ke `useOutputStore`.

## 2. Implementation

- **Lokasi Kode Sumber Utama:**
  - `frontend/components/Modals/Analyze/Descriptive/Crosstabs/`: Direktori yang berisi komponen UI.
  - `frontend/hooks/useCrosstabs.ts`: Hook yang mengelola state UI dan komunikasi dengan worker.
  - `frontend/public/workers/Crosstabs/crosstabs.worker.js`: Worker yang berisi logika untuk perhitungan Crosstabs.
