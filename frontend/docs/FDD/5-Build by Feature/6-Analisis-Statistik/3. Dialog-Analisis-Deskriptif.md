## 2. Implementation

- **Lokasi Kode Sumber Utama:**
  - `frontend/components/Modals/Analyze/Descriptive/Descriptives/`: Direktori yang berisi komponen UI.
  - `frontend/hooks/useDescriptives.ts`: Hook yang mengelola state UI dan komunikasi dengan worker.
