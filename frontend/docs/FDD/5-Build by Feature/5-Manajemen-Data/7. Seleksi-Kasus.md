## 1. Work Package

- **Fitur yang Dibangun:**
  - Mengimplementasikan dialog untuk memilih metode seleksi kasus: Kondisi Jika, Sampel Acak, atau Rentang Kasus.
  - Membuat UI builder untuk membangun ekspresi kondisional yang kompleks.
  - Membuat `selectors` service untuk mengevaluasi kondisi dan menghasilkan filter.
