# Pembangunan Fitur: Restrukturisasi Data

Dokumen ini berisi laporan pembangunan (build report) untuk Feature Set `Restrukturisasi Data`.

---

## 1. Work Package

- **Fitur yang Dibangun:**
  - Mengimplementasikan wizard untuk memandu pengguna melalui proses restrukturisasi data.
  - Menyediakan dua alur utama: mengubah variabel menjadi kasus (wide-to-long) dan kasus menjadi variabel (long-to-wide).
  - Membuat `restructureService` untuk menampung logika transformasi data yang kompleks.
  - Menghasilkan dataset baru berdasarkan hasil restrukturisasi.

## 2. Implementation

- **Lokasi Kode Sumber Utama:**
  - `frontend/components/Modals/Data/Restructure/`: Direktori yang berisi komponen UI untuk wizard.
  - `frontend/services/data/restructureService.ts`: Service yang berisi logika murni untuk operasi restrukturisasi (pivot dan unpivot).
  - `frontend/stores/useDataStore.ts`: Mengandung aksi untuk membuat dataset baru dari hasil.
