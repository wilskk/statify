## 1. Work Package

- **Fitur yang Dibangun:**
  - Mengimplementasikan dialog multi-tab yang kompleks untuk mengonfigurasi analisis kasus tidak biasa.
  - Membuat web worker (`unusualCasesWorker.js`) untuk menjalankan algoritma analisis yang intensif secara komputasi.
  - Memungkinkan pengguna untuk menyimpan hasil analisis sebagai variabel baru di dalam dataset.
  - Menghasilkan output laporan ringkas dari hasil analisis.

## 2. Implementation

- **Lokasi Kode Sumber Utama:**
  - `frontend/components/Modals/Data/UnusualCases/`: Direktori yang berisi semua komponen UI untuk dialog bertab.
  - `frontend/public/workers/unusualCasesWorker.js`: Web worker yang menampung seluruh logika analisis untuk identifikasi anomali.
  - `frontend/stores/useDataStore.ts`, `frontend/stores/useMetaStore.ts`: Berisi aksi untuk menambahkan variabel baru dari hasil analisis.
  - `frontend/stores/useOutputStore.ts`: Berisi aksi untuk menambahkan tabel ringkasan hasil ke panel output.
